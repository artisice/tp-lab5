# Описание проекта
## Personal Finance Tracker - 
это веб-приложение для личного учёта финансов, помогающее пользователям контролировать доходы и расходы. Основная функция заключается в ведении списка транзакций с категориями и отображением статистики по балансу. Приложение использует клиент-серверную архитектуру с REST API и базой данных PostgreSQL. Проект разрабатывается командой с применением гибкой методологии Scrum для итеративной поставки функционала. Приоритет отдается простоте и удобству интерфейса.
